#### カラーコレクションアプリケーション開発

【在籍期間】

- 2011 年 04⽉～2012 年 03⽉

【チーム編成】

- PM 1 人
- エンジニア 1 人
- カラリスト 1 人

【ステークホルダー】

- クライアント 1 社

【言語・フレームワーク】

- C++
- GLSL

【ツール】

- Qt
- OpenGL
- CUDA

【プロダクトが属する業界】

- 映像業界

【プロダクト概要】

- 3DCG 技術を⽤いてカラーコレクションやカラーグレーディンを⾏うためのツールを開発する
- 内部で使用している技術は会社独自のモノ
  - 3DLUT (Look Up Table)を独自技術で作成する

【チームの特徴・課題】

- 3DLUT に関する知見のあるカラリストに 3DCG の知見があるエンジニアを加えたチーム
- 3DLUT 自体は既存技術だが、Excel などを利用しており、作成や編集が扱いづらいものだった
  - 3DCG の技術を用いた独自の編集方法を用いて LUT の作成や編集しやすくした
- 3DCG の技術を用いているので、モデリングツールのように 3DLUT の編集が可能になった

【チームでの役割】

- 3DCG の知見を持っているのが自分だけだったので、要件定義から開発、顧客企業へのレクチャーも含めて担当した
- デスクトップアプリケーションの開発自体初めての経験だったが、キャッチアップしつつ、対応した
- 3DCG のモデリングなどの経験があったので、3DCG 的な操作のしやすい UI を提案した

【チームの課題と自身が工夫したこと】

- 3DLUT の知見があるが 3DCG についてはあまり知見の無い PM と、新社会人の自分だけだったので、使用するツール（Qt）等を調査し、提案しつつ進めた
- 3DCG 関係の実装経験を活かし、GPU を活用したツールに仕立てた
